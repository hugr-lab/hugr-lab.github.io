title: "SQL Server: Adventure Works"

This example demonstrates how to set up a Microsoft SQL Server database with the Adventure Works sample data as a `hugr` data source.

You can find the complete example in the [Hugr examples repository](https://github.com/hugr-lab/examples).

To read more about the SQL Server data source, see the [SQL Server / Azure SQL data source documentation](../4-engine-configuration/1-data-sources/4-sqlserver.md).
